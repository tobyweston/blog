Pi Console Lead

Set up a headless Raspberry Pi Zero using an Adafruit console lead. Step-by-step guide for Mac users on El Capitan/Sierra using the Prolific PL2303 driver.

Initial Setup

You might want to setup you‚Äôre wireless from within screen. Connecting to a non-hidden network is straight forward. Setting things up for a hidden network is a little more involved.

iface lo inet loopback
iface eth0 inet dhcp
